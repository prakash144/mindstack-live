    🚀 Linux Must-Know Concepts & Commands Guide¶

    A comprehensive Linux guide covering Beginner, Intermediate, Advanced, and Expert levels, with a focus on AWS & GCP. This guide ensures a smooth transition from fundamentals to cloud-level expertise. 🌍

    📖 Table of Contents¶

    • Beginner Level
    • Intermediate Level
    • Advanced Level
    • Expert Level
    • Summary

    🟢 Beginner Level¶

    🔹 Essential Commands¶

    • pwd – Print working directory
    • ls – List directory contents (ls -la for detailed view)
    • cd <dir> – Change directory (cd .. to move up a level)
    • mkdir <dir> – Create a new directory
    • rm <file> – Remove a file (rm -r <dir> for directories)
    • cp <src> <dest> – Copy files or directories
    • mv <src> <dest> – Move or rename files/directories
    • cat <file> – Display file contents
    • echo "text" > file.txt – Create/overwrite file with text
    • touch <file> – Create an empty file
    • man <command> – Show manual for a command
    • exit – Close terminal session

    🔹 File Permissions & Ownership¶

    • ls -l – View file permissions
    • chmod u+x <file> – Add execute permission for the user
    • chown user:group <file> – Change file ownership

    🔹 Process Management¶

    • ps aux – View running processes
    • kill <PID> – Terminate process by PID
    • kill -9 <PID> – Force terminate process
    • top / htop – Monitor system resources

    🟡 Intermediate Level¶

    🔹 File Handling & Search¶

    • find /dir -name "file*" – Search for files
    • grep "pattern" file.txt – Search for text in a file
    • grep -r "pattern" /dir – Recursively search in a directory
    • awk '{print $1}' file.txt – Process structured text
    • sed 's/old/new/g' file.txt – Replace text in a file

    🔹 Networking Commands¶

    • ping <host> – Check network connectivity
    • netstat -tulnp – Show listening ports
    • curl -I <URL> – Fetch HTTP headers
    • wget <URL> – Download a file

    🔹 Package Management¶

    • Ubuntu/Debian: apt-get install <package>
    • CentOS/RHEL: yum install <package>
    • Arch: pacman -S <package>

    🔹 Disk & Storage¶

    • df -h – Show disk usage
    • du -sh <dir> – Show directory size
    • mount /dev/sdX /mnt – Mount a disk
    • umount /mnt – Unmount a disk

    🔴 Advanced Level¶

    🔹 System Monitoring & Performance¶

    • vmstat 1 – Monitor system performance
    • iostat – Show CPU & I/O usage
    • free -m – Display memory usage
    • uptime – Show system load & uptime
    • dmesg | tail – View kernel logs

    🔹 User & Group Management¶

    • useradd -m <user> – Create a new user
    • passwd <user> – Change user password
    • usermod -aG sudo <user> – Add user to sudo group
    • groupadd <group> – Create a new group

    🔹 Shell Scripting Basics¶

    #!/bin/bash
echo "Hello, Linux!"
for i in {1..5}; do
  echo "Iteration: $i"
done

    Run it with: chmod +x script.sh && ./script.sh

    🔹 Security & Firewall¶

    • ufw enable / ufw disable – Enable/disable firewall
    • iptables -L – View firewall rules
    • fail2ban-client status – Check failed login attempts

    🔹 Container & Virtualization¶

    • docker ps – List running containers
    • docker run -it ubuntu bash – Run an Ubuntu container
    • kubectl get pods – View Kubernetes pods

    🔹 Automation & Scheduling¶

    • crontab -e – Edit cron jobs
    • Example: 0 3 * * * /path/to/script.sh # Run at 3 AM daily

    🔹 Miscellaneous¶

    • alias ll='ls -la' – Create a command alias
    • history | grep <command> – Search command history
    • nohup <command> & – Run command in background

    🔵 Expert Level¶

    🔹 System Troubleshooting¶

    • journalctl -xe – View system logs
    • strace -p <PID> – Trace system calls of a process
    • lsof -i :80 – Check processes using port 80

    🔹 Advanced File Management¶

    • tar -czvf archive.tar.gz dir/ – Compress a directory
    • rsync -av src/ dest/ – Sync files efficiently
    • scp file.txt user@host:/path/ – Securely copy file to remote

    🔹 Kubernetes & Cloud (AWS/GCP Focused)¶

    • kubectl get nodes – List Kubernetes nodes
    • kubectl logs <pod> – View pod logs
    • aws s3 ls – List S3 bucket contents
    • aws ec2 describe-instances – Get EC2 instance details
    • gcloud compute instances list – List GCP instances
    • gcloud storage buckets list – List GCP storage buckets

    📌 Summary¶

    • Beginner: Basic commands, file handling, and permissions.
    • Intermediate: Process management, networking, and automation.
    • Advanced: Security, system monitoring, scripting, and containers.
    • Expert: Troubleshooting, Kubernetes, and cloud integration.

    🚀 Master these, and you'll be a Linux Pro up to Google Senior Architect Level! 🏆
