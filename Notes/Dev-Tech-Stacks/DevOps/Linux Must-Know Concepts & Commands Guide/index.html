    üü¢ Beginner Level¶

    üîπ Essential Commands¶

    • pwd ‚Äì Print working directory
    • ls ‚Äì List directory contents (ls -la for detailed view)
    • cd <dir> ‚Äì Change directory (cd .. to move up a level)
    • mkdir <dir> ‚Äì Create a new directory
    • rm <file> ‚Äì Remove a file (rm -r <dir> for directories)
    • cp <src> <dest> ‚Äì Copy files or directories
    • mv <src> <dest> ‚Äì Move or rename files/directories
    • cat <file> ‚Äì Display file contents
    • echo "text" > file.txt ‚Äì Create/overwrite file with text
    • touch <file> ‚Äì Create an empty file
    • man <command> ‚Äì Show manual for a command
    • exit ‚Äì Close terminal session

    üîπ File Permissions & Ownership¶

    • ls -l ‚Äì View file permissions
    • chmod u+x <file> ‚Äì Add execute permission for the user
    • chown user:group <file> ‚Äì Change file ownership

    üîπ Process Management¶

    • ps aux ‚Äì View running processes
    • kill <PID> ‚Äì Terminate process by PID
    • kill -9 <PID> ‚Äì Force terminate process
    • top / htop ‚Äì Monitor system resources

    üü° Intermediate Level¶

    üîπ File Handling & Search¶

    • find /dir -name "file*" ‚Äì Search for files
    • grep "pattern" file.txt ‚Äì Search for text in a file
    • grep -r "pattern" /dir ‚Äì Recursively search in a directory
    • awk '{print $1}' file.txt ‚Äì Process structured text
    • sed 's/old/new/g' file.txt ‚Äì Replace text in a file

    üîπ Networking Commands¶

    • ping <host> ‚Äì Check network connectivity
    • netstat -tulnp ‚Äì Show listening ports
    • curl -I <URL> ‚Äì Fetch HTTP headers
    • wget <URL> ‚Äì Download a file

    üîπ Package Management¶

    • Ubuntu/Debian: apt-get install <package>
    • CentOS/RHEL: yum install <package>
    • Arch: pacman -S <package>

    üîπ Disk & Storage¶

    • df -h ‚Äì Show disk usage
    • du -sh <dir> ‚Äì Show directory size
    • mount /dev/sdX /mnt ‚Äì Mount a disk
    • umount /mnt ‚Äì Unmount a disk

    üî¥ Advanced Level¶

    üîπ System Monitoring & Performance¶

    • vmstat 1 ‚Äì Monitor system performance
    • iostat ‚Äì Show CPU & I/O usage
    • free -m ‚Äì Display memory usage
    • uptime ‚Äì Show system load & uptime
    • dmesg | tail ‚Äì View kernel logs

    üîπ User & Group Management¶

    • useradd -m <user> ‚Äì Create a new user
    • passwd <user> ‚Äì Change user password
    • usermod -aG sudo <user> ‚Äì Add user to sudo group
    • groupadd <group> ‚Äì Create a new group

    üîπ Shell Scripting Basics¶

    #!/bin/bash
echo "Hello, Linux!"
for i in {1..5}; do
  echo "Iteration: $i"
done

    Run it with: chmod +x script.sh && ./script.sh

    üîπ Security & Firewall¶

    • ufw enable / ufw disable ‚Äì Enable/disable firewall
    • iptables -L ‚Äì View firewall rules
    • fail2ban-client status ‚Äì Check failed login attempts

    üîπ Container & Virtualization¶

    • docker ps ‚Äì List running containers
    • docker run -it ubuntu bash ‚Äì Run an Ubuntu container
    • kubectl get pods ‚Äì View Kubernetes pods

    üîπ Automation & Scheduling¶

    • crontab -e ‚Äì Edit cron jobs
    • Example: 0 3 * * * /path/to/script.sh # Run at 3 AM daily

    üîπ Miscellaneous¶

    • alias ll='ls -la' ‚Äì Create a command alias
    • history | grep <command> ‚Äì Search command history
    • nohup <command> & ‚Äì Run command in background

    üîµ Expert Level¶

    üîπ System Troubleshooting¶

    • journalctl -xe ‚Äì View system logs
    • strace -p <PID> ‚Äì Trace system calls of a process
    • lsof -i :80 ‚Äì Check processes using port 80

    üîπ Advanced File Management¶

    • tar -czvf archive.tar.gz dir/ ‚Äì Compress a directory
    • rsync -av src/ dest/ ‚Äì Sync files efficiently
    • scp file.txt user@host:/path/ ‚Äì Securely copy file to remote

    üîπ Kubernetes & Cloud (AWS/GCP Focused)¶

    • kubectl get nodes ‚Äì List Kubernetes nodes
    • kubectl logs <pod> ‚Äì View pod logs
    • aws s3 ls ‚Äì List S3 bucket contents
    • aws ec2 describe-instances ‚Äì Get EC2 instance details
    • gcloud compute instances list ‚Äì List GCP instances
    • gcloud storage buckets list ‚Äì List GCP storage buckets

    üìå Summary¶

    • Beginner: Basic commands, file handling, and permissions.
    • Intermediate: Process management, networking, and automation.
    • Advanced: Security, system monitoring, scripting, and containers.
    • Expert: Troubleshooting, Kubernetes, and cloud integration.
