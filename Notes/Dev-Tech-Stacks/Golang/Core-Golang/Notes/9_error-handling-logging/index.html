    🚨 Error Handling & Logging in Go¶

    📌 Overview¶

    Error handling in Go follows a simple, explicit approach using error values. Logging is essential for debugging and monitoring applications.

    🛠️ Error Handling in Go¶

    ✅ Standard Error Handling¶

    package main
import (
    "errors"
    "fmt"
)

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero is not allowed")
    }
    return a / b, nil
}

func main() {
    result, err := divide(10, 0)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println("Result:", result)
}

    🎯 Using fmt.Errorf for Contextual Errors¶

    import "fmt"
err := fmt.Errorf("failed to process request: %w", originalErr)

    🎭 Custom Error Types¶

    import "fmt"

type CustomError struct {
    Code    int
    Message string
}

func (e *CustomError) Error() string {
    return fmt.Sprintf("[Error %d]: %s", e.Code, e.Message)
}

func main() {
    err := &CustomError{Code: 404, Message: "Resource Not Found"}
    fmt.Println(err)
}

    📜 Logging in Go¶

    📝 Using the Standard log Package¶

    import "log"

func main() {
    log.Println("This is an info message")
    log.Fatal("This will exit the program")
}

    🚀 Advanced Logging with logrus¶

    import (
    "github.com/sirupsen/logrus"
)

func main() {
    log := logrus.New()
    log.SetLevel(logrus.InfoLevel)
    log.WithFields(logrus.Fields{
        "user": "admin",
    }).Info("User logged in")
}

    📊 Structured Logging with zap¶

    import (
    "go.uber.org/zap"
)

func main() {
    logger, _ := zap.NewProduction()
    defer logger.Sync()
    logger.Info("Application started", zap.String("version", "1.0.0"))
}

    ✅ Best Practices¶

    ✔️ Always return errors explicitly (error type).
    ✔️ Use fmt.Errorf to wrap errors with context.
    ✔️ Implement custom error types where necessary.
    ✔️ Prefer structured logging (logrus, zap) for production.
    ✔️ Log errors with stack trace for better debugging.
    ✔️ Avoid panic in production code unless absolutely necessary.
