    📌 Response Formatting & Status Codes – Gin Framework 🚀¶

    🔹 Sending JSON Response 📝¶

    c.JSON(200, gin.H{
    "message": "Success",
    "status": 200,
})

    🔹 Setting Custom Status Codes 📟¶

    c.JSON(201, gin.H{"message": "Created"}) // HTTP 201 - Created
c.JSON(400, gin.H{"error": "Bad Request"}) // HTTP 400 - Bad Request
c.JSON(404, gin.H{"error": "Not Found"}) // HTTP 404 - Not Found
c.JSON(500, gin.H{"error": "Internal Server Error"}) // HTTP 500 - Internal Error

    🔹 Sending Plain Text Response 🏷️¶

    c.String(200, "Hello, World!")

    🔹 Sending HTML Response 🌐¶

    c.HTML(200, "<h1>Welcome to Gin</h1>")

    🔹 Redirecting Requests 🔀¶

    c.Redirect(302, "https://example.com") // Temporary Redirect

    🔹 Streaming Response 📡¶

    c.Stream(func(w io.Writer) bool {
    w.Write([]byte("Streaming data...\n"))
    return true
})

    🔹 Abort with Status 🚨¶

    c.AbortWithStatus(403) // Abort request with HTTP 403 - Forbidden

    🔹 Response with Headers 📜¶

    c.Header("Content-Type", "application/json")
c.JSON(200, gin.H{"message": "Headers set successfully"})

    🔹 Common HTTP Status Codes 🚦¶

    Code Meaning
    200 OK ✅
    201 Created 🆕
    204 No Content 🚫
    400 Bad Request ⚠️
    401 Unauthorized 🔒
    403 Forbidden 🚫
