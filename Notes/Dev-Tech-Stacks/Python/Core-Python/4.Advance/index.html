    Advanced Python Notes - Quick Revision¶

    1. Build Your Own REST API with Python¶

    • Example: from flask import Flask, jsonify; app = Flask(__name__); @app.route('/api', methods=['GET']); def api(): return jsonify({'message': 'Hello, World!'})

    2. Build Your Own Blog¶

    • Use Flask or Django. Example: from flask import Flask; app = Flask(__name__); @app.route('/'); def home(): return "Welcome to My Blog!"

    3. Databases with SQLite¶

    • Example: import sqlite3; conn = sqlite3.connect('example.db'); cursor = conn.cursor(); cursor.execute('CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)')

    4. Dataframe Inspection¶

    • Example: import pandas as pd; df = pd.DataFrame(data); print(df.head())

    5. Data Cleaning¶

    • Example: df['column'] = df['column'].fillna(0)

    6. Sorting Values in Dataframes¶

    • Example: df = df.sort_values(by='column')

    7. Arithmetic Operations with Pandas¶

    • Example: df['sum'] = df['col1'] + df['col2']

    8. Creating Pivot Tables¶

    • Example: pivot = df.pivot_table(values='sales', index='region', columns='month', aggfunc='sum')

    9. Chaining Functions¶

    • Example: df = df.dropna().sort_values(by='column').reset_index(drop=True)

    10. Smoothing Time Series Data¶

    • Example: df['smoothed'] = df['column'].rolling(window=3).mean()

    11. Creating Line Charts with Matplotlib¶

    • Example: import matplotlib.pyplot as plt; plt.plot(x, y); plt.show()

    12. Using Jupyter Notebook¶

    • Use %matplotlib inline for inline plotting.

    13. HTML Markdown¶

    • Example: **Bold** for bold text; *Italic* for italics.

    14. Creating Scatterplots with Matplotlib¶

    • Example: plt.scatter(x, y); plt.show()

    15. Relational Database Schemas¶

    • Define tables and relationships with primary/foreign keys.

    16. Descriptive Statistics¶

    • Example: df.describe()

    17. Creating Charts with Plotly¶

    • Example: import plotly.express as px; fig = px.bar(df, x='x', y='y'); fig.show()

    18. Creating NumPy NDArrays¶

    • Example: import numpy as np; arr = np.array([1, 2, 3])

    19. Array Slicing and Subsetting¶

    • Example: arr[1:3]

    20. Matrix Multiplication¶

    • Example: np.dot(matrix1, matrix2)

    21. Bitwise and Operators in Pandas¶

    • Example: df[(df['col1'] > 0) & (df['col2'] < 5)]

    22. Creating Bubble Charts with Seaborn¶

    • Example: import seaborn as sns; sns.scatterplot(data=df, x='x', y='y', size='size')

    23. Running Regressions with Scikit-Learn¶

    • Example: from sklearn.linear_model import LinearRegression; model = LinearRegression(); model.fit(X, y)

    24. Non-Parametric Regression¶

    • Use libraries like statsmodels or sklearn.

    25. Students T-Tests and Histograms with Scikit-Learn¶

    • Example: from scipy.stats import ttest_ind; ttest_ind(group1, group2)

    26. Multi-Variable Regression¶

    • Example: from sklearn.linear_model import LinearRegression; model = LinearRegression(); model.fit(X, y)

    27. Log Transformations¶

    • Example: import numpy as np; df['log_column'] = np.log(df['column'])

    28. Residuals Analysis¶

    • Example: residuals = y - model.predict(X); plt.hist(residuals); plt.show()
