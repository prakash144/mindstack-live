    Python Beginner Notes - Quick Revision¶

    1. Variables in Python¶

    • Store data with a name. Example: x = 5, name = "Alice"

    2. String Manipulation¶

    • Use methods like .upper(), .lower(), .strip(), .replace(). Example: message.upper() → HELLO

    3. Input and Print Functions¶

    • input() for user input, print() for output. Example: name = input("Enter your name: "); print(f"Hello, {name}")

    4. Variable Naming Rules¶

    • Must start with a letter or _; no spaces or special characters; case-sensitive. Example: _valid_name = 42

    5. Mathematical Operations in Python¶

    • Basic: +, -, *, /, %, **. Example: 5 ** 2 → 25

    6. Data Types¶

    • Common types: int, float, str, list, dict, bool.

    7. Converting Types¶

    • Use int(), str(), float(). Example: age = int("25")

    8. Conditionals: IF/ELIF/ELSE¶

    • Use for decision-making. Example:
    • if x > 0: print("Positive")
    • elif x == 0: print("Zero")
    • else: print("Negative")

    9. Logical Operators¶

    • Use and, or, not. Example: if x > 0 and y > 0: print("Both are positive")

    10. Randomisation¶

    • Use the random module. Example: import random; print(random.randint(1, 10))

    11. Error Handling¶

    • Use try and except for managing errors. Example:
    • try: print(10 / 0)
    • except ZeroDivisionError: print("Cannot divide by zero")

    12. Functions¶

    • Define reusable blocks of code. Example: def greet(name): return f"Hello, {name}"

    13. For Loops¶

    • Iterate over sequences. Example: for i in range(5): print(i)

    14. Code Blocks and Indentation¶

    • Python relies on indentation to define blocks. Example: if True: print("Indented block")

    15. While Loops¶

    • Run as long as condition is True. Example: while x < 5: x += 1

    16. Flowchart Programming¶

    • Visual representation of logic; helps in planning loops and conditions.

    17. Positional and Keyword Arguments¶

    • Positional: Order matters. Keyword: Name-value pairs. Example: greet(age=25, name="Alice")

    18. Python Dictionaries and Lists¶

    • List: Ordered, mutable. Example: fruits = ["apple", "banana"]
    • Dict: Key-value pairs. Example: data = {"name": "Alice", "age": 25}

    19. Nested Collections¶

    • Collections inside collections. Example: data = {"users": [{"name": "Alice"}, {"name": "Bob"}]}

    20. Returning Functions¶

    • Functions can return values. Example: def add(a, b): return a + b

    21. Return vs. Print¶

    • Return: Gives a value back.
    • Print: Displays output.

    22. Doc Strings vs. Comments¶

    • Doc Strings: Describe function behavior; enclosed in """.
    • Comments: Inline explanations using #.

    23. Scope and Local/Global Variables¶

    • Local: Inside a function. Global: Declared outside functions. Example:
    • x = 10
    • def func(): global x; x = 20

    24. Debugging Techniques¶

    • Use print() or tools like pdb. Example with pdb: import pdb; pdb.set_trace()
