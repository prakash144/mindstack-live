    Intermediate Python Notes - Quick Revision¶

    1. Local Development Environment Setup¶

    • Install Python, IDEs (PyCharm/VSCode), and manage dependencies with pip.

    2. PyCharm Tips and Tricks¶

    • Use debugging tools, code inspections, and shortcuts like Ctrl+D (duplicate line).

    3. Python Object-Oriented Programming (OOP)¶

    • Core principles: Encapsulation, Inheritance, Polymorphism, Abstraction.

    4. Creating Classes in Python¶

    • Use class keyword. Example: 
    class Person: 
   def __init__(self, name):
     self.name = name

    5. Using External Python Modules/Import¶

    • Use import or from ... import. Example: import math; print(math.sqrt(16))

    6. Getting/Setting Attributes¶

    • Access or modify object properties using self.attribute. Example: obj.name = "Alice"

    7. Python Methods¶

    • Functions defined inside a class. Example:
      class Dog: def bark(self): return "Woof"

    8. Class Initializers¶

    • Use __init__ for initializing object attributes. Example:
      class Car: def __init__(self, brand): self.brand = brand

    9. Module Aliasing¶

    • Shorten module names using as. Example: import numpy as np

    10. Optional, Required, and Default Parameters¶

    • Example: def greet(name, age=25): return f"{name} is {age} years old"

    11. Event Listeners¶

    • Use libraries like pygame to listen to events. Example:
      if event.type == pygame.KEYDOWN: print("Key pressed")

    12. Python Instances and State¶

    • Instances store unique data in object attributes. Example:
      p1 = Person("Alice"); p2 = Person("Bob")

    13. Python Turtle¶

    • Graphics with turtle module. Example:
      import turtle; t = turtle.Turtle(); t.forward(100)

    14. Game Development with Python and OOP¶

    • Use pygame for games. Example:
      import pygame; pygame.init(); screen = pygame.display.set_mode((400, 300))

    15. Python Inheritance¶

    • Extend a base class. Example:
      class Dog(Animal): pass

    16. Python Slice Function¶

    • Extract sequences with slicing. Example:
      list = [1, 2, 3, 4]; print(list[1:3])

    17. File I/O (Reading/Writing Local Files)¶

    • Example:
      with open('file.txt', 'w') as f: f.write('Hello')

    18. File Directories¶

    • Use os module to manage directories. Example:
      import os; os.mkdir('new_folder')

    19. Reading/Writing to CSV¶

    • Example:
      import csv; with open('file.csv') as f: reader = csv.reader(f); for row in reader: print(row)

    20. Introduction to the Pandas Framework¶

    • Data manipulation library. Example:
      import pandas as pd; df = pd.read_csv('file.csv'); print(df.head())

    21. List Comprehensions¶

    • Compact list creation. Example: [x**2 for x in range(5)]

    22. Dictionary Comprehensions¶

    • Example: {x: x**2 for x in range(5)}

    23. Packing/Unpacking Functions in Python¶

    • Use * and **. Example:
      def add(*args): return sum(args); print(add(1, 2, 3))

    24. Creating Desktop GUI Apps with Tkinter¶

    • Example:
      import tkinter as tk; root = tk.Tk(); root.mainloop()

    25. Strongly Dynamic Typing¶

    • Python determines type at runtime but enforces type constraints.

    26. Error Handling and Exceptions¶

    • Handle errors with try and except. Example:
      try: x = 1 / 0 except ZeroDivisionError: print("Error!")

    27. Try/Except/Raise¶

    • Use raise to trigger exceptions. Example:
      if x < 0: raise ValueError("Negative value!")

    28. Working with JSONs¶

    • Example:
      import json; data = json.loads('{"key": "value"}'); print(data)

    29. Local Persistence¶

    • Save data locally with pickle or databases.

    30. Sending Email with Python and SMTP¶

    • Example:
      import smtplib; server = smtplib.SMTP('smtp.example.com', 587); server.sendmail(from_addr, to_addr, message)

    31. Working with Date and Time¶

    • Use datetime module. Example:
      from datetime import datetime; print(datetime.now())

    32. Hosting Python Code Online with PythonAnywhere¶

    • Deploy code on platforms like PythonAnywhere for web hosting.
