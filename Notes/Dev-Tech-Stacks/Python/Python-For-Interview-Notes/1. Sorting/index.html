    📌 Python Sorting: A Complete Guide¶

    (Comprehensive guide for Python sorting methods. 🎯 Focus: Implementation details and practical usage)

    🔄 Basic Sorting¶

    📝 .sort() Method¶

    • ✨ Basic Usage: python elements = [5, 3, 6, 2, 1] elements.sort() # [1, 2, 3, 5, 6]
    • 🔑 Key Features:
      • Modifies list in-place
      • Returns None
      • Default ascending order

    🔤 String Sorting¶

    words = ["grape", "apple", "banana", "orange"]
words.sort()  # ['apple', 'banana', 'grape', 'orange']
    • ⚡ Uses lexicographical order
    • Case-sensitive by default

    ⚙️ Method Parameters¶

    🔧 Function Signature¶

    def sort(key=None, reverse=False) -> None

    🔄 Reverse Sorting¶

    • 📝 Using reverse parameter: python elements = [5, 3, 6, 2, 1] elements.sort(reverse=True) # [6, 5, 3, 2, 1]
    • 🔄 Alternative approach: python elements.sort() elements.reverse()

    🎯 Custom Sorting¶

    🔑 Using Functions¶

    def get_word_length(word: str) -> int:
    return len(word)

words = ["apple", "banana", "kiwi"]
words.sort(key=get_word_length)

    λ Lambda Functions¶

    # Sort by length
words.sort(key=lambda word: len(word))

# Sort by last character
words.sort(key=lambda x: x[-1])

    🆕 Sorted Function¶

    📝 Basic Usage¶

    # Creates new sorted list
original = [3, 1, 4]
new_list = sorted(original)  # original unchanged

    🔧 Advanced Usage¶

    # With reverse
sorted(numbers, reverse=True)

# With key function
sorted(numbers, key=abs)  # Sort by absolute value

    💡 Implementation Notes¶

    🔍 Key Differences¶

    • .sort():
      • In-place modification
      • List method only
      • Returns None
    • sorted():
      • Creates new list
      • Works with any iterable
      • Returns sorted list

    ⚠️ Common Pitfalls¶

    # WRONG ❌
sorted_list = list.sort()  # Returns None!

# CORRECT ✅
list.sort()
# OR
sorted_list = sorted(list)

    🚀 Quick Reference¶

    # Basic sorting
list.sort()                    # In-place, ascending
sorted(list)                   # New list, ascending

# Reverse sorting
list.sort(reverse=True)        # In-place, descending
sorted(list, reverse=True)     # New list, descending

# Custom sorting
list.sort(key=len)            # By length
list.sort(key=lambda x: x[1])  # By second element

    🎯 Key Takeaway: Choose .sort() for in-place modification and sorted() when you need to preserve the original list or work with other iterables.
