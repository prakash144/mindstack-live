    1. New Process ‚Üí Placed in Ready Queue.

    üéØ Schedulers¶

    üõ†Ô∏è Types of Schedulers¶

    1. Long-Term Scheduler (Job Scheduler) üïí

      • Temporarily removes processes from memory (swapping).
      • Manages memory overcommitment.
      • Reintroduces processes when resources are available.
