    📡 Network Cabling and Connectors¶

    Network cabling provides the physical medium for data transfer, while connectors link cables to devices for seamless signal transmission.

    📌 Table of Contents¶

    • 📢 Importance
    • 🔗 Twisted Pair Cables

    • 🔌 Coaxial Cables
    • 🖧 Network Devices
    • ⚖️ Comparison of Network Devices
    • ✅ Quick Revision Summary

    📢 Importance¶

    • ✔️ Foundation for network communication 🏗️
    • ✔️ Determines speed, reliability, and scalability ⚡
    • ✔️ Impacts performance & security 🔒

    🔗 Twisted Pair Cables¶

    Two insulated copper wires twisted to reduce electromagnetic interference (EMI) and crosstalk.

    🔍 Types¶

    Type Features Use Case
    UTP (Unshielded Twisted Pair) No shielding, cost-effective Common in Ethernet networks
    STP (Shielded Twisted Pair) Shielded for better interference resistance Industrial/high-noise environments

    ✅ Pros & ❌ Cons¶

    ✔️ Cost-effective, easy to install ✔️ Suitable for short to medium distances ❌ Susceptible to interference (UTP) ❌ Lower bandwidth than fiber optics

    🔌 Common Connectors¶

    • RJ45 – Standard for Ethernet networking
    • RJ11 – Used for telephone connections

    💡 Fiber Optic Cables¶

    Transmit data as light signals through glass or plastic fibers for high-speed, long-distance communication.

    🔍 Types¶

    Type Features Use Case
    SMF (Single-Mode Fiber) Uses a single light wave Long-distance communication (ISP backbones)
    MMF (Multi-Mode Fiber) Uses multiple light waves Short-distance networks (campuses, buildings)

    ✅ Pros & ❌ Cons¶

    ✔️ High bandwidth & speed 🚀 ✔️ Immune to EMI 🛡️ ✔️ Ideal for long distances 🌍 ❌ Expensive to install & maintain 💰 ❌ Fragile compared to copper cables ⚠️

    🔌 Common Connectors¶

    • LC (Lucent Connector) – High-density networks, data centers
    • SC (Subscriber Connector) – Enterprise networks, push-pull mechanism
    • ST (Straight Tip) – Older industrial networks

    🔌 Coaxial Cables¶

    A central conductor with shielding for low-interference data transmission.

    🔍 Uses¶

    ✔️ Cable TV, broadband internet ✔️ Older Ethernet networks

    ✅ Pros & ❌ Cons¶

    ✔️ EMI-resistant due to shielding ✔️ Durable for moderate distances ❌ Bulkier and less flexible than twisted pair ❌ Limited speed and bandwidth

    🔌 Common Connectors¶

    • BNC (Bayonet Neill–Concelman) – Security cameras, legacy Ethernet
    • F-Type – Cable TV, broadband

    🖧 Network Devices¶

    Manage communication, routing, and data transfer in networks.

    📌 Importance¶

    ✔️ Facilitates data routing & switching 🔀 ✔️ Improves network performance & scalability 📊 ✔️ Ensures security & data integrity 🔒

    🔍 Devices Overview¶

    📡 Router¶

    • Routes data between different networks 🌎
    • Provides firewall, NAT, and security 🔐
    • Use Case: Homes, offices, ISPs

    🔄 Switch¶

    • Connects multiple devices within a network 🏢
    • Layer 2 (Data Link Layer) device 📡
    • Use Case: LANs, servers, printers

    ⚡ Hub¶

    • Broadcasts data to all connected devices 📢
    • Layer 1 (Physical Layer) device 🛠️
    • Use Case: Small networks

    🌉 Bridge¶

    • Connects and segments networks 🔗
    • Filters traffic based on MAC addresses 🎛️
    • Use Case: LAN expansion

    📶 Repeater¶

    • Amplifies & regenerates signals to extend network range 📡
    • Layer 1 (Physical Layer) device 🔋
    • Use Case: Long-distance communication, Wi-Fi extension

    ⚖️ Comparison of Network Devices¶

    Device Layer Key Function Use Case
    Router Network Layer Routes data between networks Internet access, WANs
    Switch Data Link Layer Forwards data to specific devices LAN communication
    Hub Physical Layer Broadcasts data to all devices Small networks
    Bridge Data Link Layer Connects and segments networks LAN expansion
    Repeater Physical Layer Amplifies and regenerates signals Extending network range

    ✅ Quick Revision Summary¶

    • ✔️ Twisted Pair Cables – Cost-effective, used in Ethernet networks (UTP, STP).
    • ✔️ Fiber Optic Cables – High-speed, long-distance (SMF, MMF), used in ISPs, data centers.
    • ✔️ Coaxial Cables – Used in cable TV, older networks, moderate speeds.
    • ✔️ Router – Routes data between networks, connects LANs to the Internet.
    • ✔️ Switch – Connects multiple devices in LANs, reducing congestion.
    • ✔️ Hub – Basic, inefficient, used in small networks.
    • ✔️ Bridge – Segments networks for better traffic management.
    • ✔️ Repeater – Amplifies signals for long-distance transmission.
