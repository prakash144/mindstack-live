    ðŸš€ 1. String in Java¶

    • String is a sequence of characters represented as an object of the String class.
    • Immutable: Once a String object is created, its value cannot be changed.
    • Stored in the String Pool: Improves memory efficiency by storing identical string literals only once.
    • Syntax:
    String str1 = "Hello";  // String literal
String str2 = new String("World");  // String object

    ðŸ”¥ 2. Immutability in String¶

    • Definition: When a String object is created, its value cannot be modified.
    • Why immutable?
      • Thread-safety: Multiple threads can safely access String objects without synchronization.
      • Security: Immutable strings are used in sensitive operations (e.g., database credentials).
      • String Pooling: Prevents duplicate String objects, improving memory usage.
    • Example:
    String str = "Java";
str.concat(" Rocks");    // Creates a new String object, doesn't modify original
System.out.println(str);  // Output: Java

    ðŸ”¥ 3. String Interning¶

    • Definition: String interning is the process of storing only one copy of each distinct string value in the String Pool.
    • When you create a string using literals, Java automatically interns them.
    • Use String.intern() to manually intern a string.
    • Example:
    String str1 = "Java";   // Interned string (in the pool)
String str2 = new String("Java").intern();  // Interns the object
System.out.println(str1 == str2);  // true (both refer to the same pool object)

    ðŸ”¥ 4. String Manipulation Methods¶

    • Length & Extraction:
    String str = "Hello, Java!";
System.out.println(str.length());           // 12
System.out.println(str.charAt(1));          // e
System.out.println(str.substring(0, 5));    // Hello
    • Concatenation:
    String s1 = "Hello";
String s2 = " World";
System.out.println(s1.concat(s2));   // Hello World
    • Comparison:
    String str1 = "Java";
String str2 = "JAVA";
System.out.println(str1.equals(str2));       // false
System.out.println(str1.equalsIgnoreCase(str2));  // true
    • String Replacement:
    String str = "Java is awesome";
System.out.println(str.replace("awesome", "powerful"));
// Output: Java is powerful
    • Trimming & Formatting:
    String str = "  Hello  ";
System.out.println(str.trim());             // "Hello" (removes leading & trailing spaces)
System.out.println(String.format("%s %d", "Age:", 30));  // Age: 30

    ðŸš€ 5. StringBuilder in Java¶

    • StringBuilder is a mutable sequence of characters used for efficient string manipulation.
    • Faster performance compared to String for frequent modifications (append, insert).
    • Not thread-safe, but faster than StringBuffer.
    • Syntax:
    StringBuilder sb = new StringBuilder("Hello");
sb.append(" World");
System.out.println(sb);  // Hello World
