    üîÑ Control Flow Statements in Java¶

    ‚öôÔ∏è 1. Introduction to Control Flow Statements¶

    • Control flow statements determine the order of execution in a program.
    • Types of control flow statements:
      • Conditional Statements: if, if-else, if-else-if, switch
      • Looping Statements: for, while, do-while
      • Jump Statements: break, continue, return

    ‚úÖ 2. Conditional Statements¶

    üî• 2.1 if-else Statement¶

    • Used for decision-making based on conditions.
    • Syntax:
    if (condition) {
    // Executes if the condition is true
} else {
    // Executes if the condition is false
}
    • Example:
    int num = 10;
if (num > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Negative");
}
    • Nested if-else:
    int a = 10, b = 20;
if (a > b) {
    System.out.println("a is greater");
} else if (a < b) {
    System.out.println("b is greater");
} else {
    System.out.println("Both are equal");
}

    üî• 2.2 switch-case Statement¶

    • Used for multiple condition checking.
    • More efficient than a long chain of if-else statements.
    • Syntax:
    switch(expression) {
    case value1:
        // Code to execute if expression == value1
        break;
    case value2:
        // Code to execute if expression == value2
        break;
    default:
        // Code to execute if no case matches
}
    • Example:
    int day = 3;
switch(day) {
    case 1: System.out.println("Monday"); break;
    case 2: System.out.println("Tuesday"); break;
    case 3: System.out.println("Wednesday"); break;
    default: System.out.println("Invalid Day");
}
    • Enhancements in Java 12+:
      • Switch Expressions introduced in Java 12, simplify the syntax:
    String result = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3 -> "Wednesday";
    default -> "Invalid Day";
};
System.out.println(result);

    üîÅ 3. Looping Statements¶

    üî• 3.1 for Loop¶

    • Used to iterate over a block of code a fixed number of times.
    • Syntax:
    for(initialization; condition; increment/decrement) {
    // Code to execute repeatedly
}
    • Example:
    for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}
    • Enhanced for-loop (for-each) introduced in Java 5, used with arrays and collections:
    int[] nums = {1, 2, 3, 4, 5};
for (int num : nums) {
    System.out.println(num);
}

    üî• 3.2 while Loop¶

    • Repeats the block of code as long as the condition is true.
    • Syntax:
    while (condition) {
    // Code to execute repeatedly
}
    • Example:
    int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}

    üî• 3.3 do-while Loop¶

    • Similar to while loop but executes at least once.
    • Syntax:
    do {
    // Code to execute repeatedly
} while (condition);
    • Example:
    int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 5);

    üöÄ 4. Jump Statements¶

    üî• 4.1 break Statement¶

    • Used to terminate a loop or switch-case immediately.
    • Example:
    for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;   // Exit the loop
    }
    System.out.println(i);
}

    üî• 4.2 continue Statement¶

    • Skips the current iteration and continues with the next one.
    • Example:
    for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;   // Skip iteration when i == 3
    }
    System.out.println(i);
}

    üî• 4.3 return Statement¶

    • Exits from the current method and returns a value.
    • Example:
    public int add(int a, int b) {
    return a + b;  // Returns the sum of a and b
}

    ‚úÖ 5. Key Takeaways¶

    • if-else: Used for binary decisions.
    • switch-case: Better for multiple condition checks.
    • for loop: When the number of iterations is known.
    • while loop: When the condition is checked before executing the loop.
    • do-while loop: Ensures at least one iteration.
    • break and continue: Modify loop execution.
    • return: Exits from the current method and optionally returns a value.

    üöÄ 6. Best Practices & Tips¶

    • ‚úÖ Use switch-case for readability when handling multiple conditions.
    • ‚úÖ Use Enhanced for-loops when iterating over collections or arrays.
    • ‚úÖ Avoid infinite loops with proper termination conditions.
    • ‚úÖ Use break and continue cautiously to avoid unexpected behavior.
    • ‚úÖ Always initialize loop variables properly to prevent errors.
    • ‚úÖ Use Java 12+ switch expressions for cleaner, concise code.

    üî• 7. Interview Tips¶

    • Expect questions on loop optimizations and nested loops.
    • Be familiar with switch expressions (Java 12+) as they are commonly asked.
    • Understand the difference between while and do-while loops, and when to use each.
    • Know how to use break and continue effectively in complex loops.
