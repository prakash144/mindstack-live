    🔥 Functional Interfaces in Java¶

    🧠 1. What is a Functional Interface?¶

    • An interface with exactly one abstract method.
    • Can have multiple default or static methods.
    • Enables usage of lambda expressions and method references.
    • Annotated with @FunctionalInterface for compile-time validation (optional but recommended).
    @FunctionalInterface
public interface MyFunctionalInterface {
    void execute();
}

    ⚙️ 2. Built-in Functional Interfaces (java.util.function)¶

    Java 8 introduced commonly used functional interfaces in the java.util.function package:

    🔹 Interface ✅ Method Signature 🎯 Purpose 🔁 Input 🎁 Output
    Consumer<T> void accept(T t) Performs action on a value, returns nothing 1 void
    Supplier<T> T get() Supplies a value, no input 0 1
    Predicate<T> boolean test(T t) Tests a condition on input, returns boolean 1 boolean
    Function<T,R> R apply(T t) Transforms input to output 1 1

    🔍 3. Use Cases & Examples¶

    ✅ Consumer: Side-effects like logging, printing¶

    Consumer<String> printer = msg -> System.out.println(msg);
printer.accept("Hello 👋");

    ✅ Supplier: Lazy evaluation, dependency injection¶

    Supplier<Double> randomSupplier = () -> Math.random();
System.out.println(randomSupplier.get());

    ✅ Predicate: Filtering, validations¶

    Predicate<Integer> isEven = n -> n % 2 == 0;
System.out.println(isEven.test(4)); // true

    ✅ Function: Mapping, transformations¶

    Function<String, Integer> lengthFunc = s -> s.length();
System.out.println(lengthFunc.apply("Java")); // 4

    🚀 4. Advanced Usage Tips (for experienced developers)¶

    • 🧩 Composing functions: Chain using andThen(), compose() (for Function)
    Function<String, String> trim = String::trim;
Function<String, Integer> length = String::length;
Function<String, Integer> pipeline = trim.andThen(length);
    • 🔗 Chaining predicates:
    Predicate<String> startsWithJ = s -> s.startsWith("J");
Predicate<String> lengthCheck = s -> s.length() > 3;
Predicate<String> combined = startsWithJ.and(lengthCheck);
    • 💡 Functional interfaces + Streams = expressive, readable code
    List<String> names = List.of("John", "Jill", "Bob");
names.stream()
     .filter(s -> s.startsWith("J"))
     .map(String::toUpperCase)
     .forEach(System.out::println);

    📌 Summary¶

    • Functional interfaces enable clean, declarative, and functional programming in Java.
    • Core to building blocks of lambda expressions, streams, and modern Java patterns.
    • Use with caution in complex logic: prefer clarity over cleverness.

    🔥 Pro Tip: You can create custom functional interfaces to model your domain-specific operations more clearly when built-in ones aren't expressive enough.
