    ðŸš€ 4. Deque Interface¶

    • Part of: java.util package.
    • Implements: Queue interface.
    • Allows insertion/removal from both ends.
    • Key Methods:
      • addFirst(E e) / offerFirst(E e) â†’ Inserts at the front.
      • addLast(E e) / offerLast(E e) â†’ Inserts at the rear.
      • removeFirst() / pollFirst() â†’ Removes from the front.
      • removeLast() / pollLast() â†’ Removes from the rear.
      • getFirst() / peekFirst() â†’ Retrieves the front element.
      • getLast() / peekLast() â†’ Retrieves the rear element.

System.out.println(deque.pollLast());    // Removes and prints: Banana
