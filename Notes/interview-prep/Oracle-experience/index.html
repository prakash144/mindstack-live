    🔐 Identity Governance: Concepts Simplified¶

    • 🎯 Purpose:

      • Show who has access to what
      • Enforce and automate access control
      • Support on-demand and contextual access assignment

    • 🛡️ Why VPD?

      • Enables fine-grained access control.
      • Ensures data-level security in multi-tenant systems.

    • 🚀 Framework Used:

      • Dropwizard: Lightweight Java framework used for rapid development of high-performance RESTful services.

    🧠 Key Learnings¶

    • Mastery in RESTful API development using Java + Dropwizard.
    • Hands-on with secure identity governance models and OCI.
    • Improved proficiency in DevOps automation, CI/CD, and cross-functional collaboration.

    💬 "My work in Oracle's governance team gave me deep insights into secure system architecture, identity access control, and cloud-native development."

    🌟 Why It Matters¶

    • Built systems that scale securely and comply with governance policies.
    • Directly impacted cloud product usability and security infrastructure.
    • Gained valuable experience working on enterprise-scale cloud solutions.
